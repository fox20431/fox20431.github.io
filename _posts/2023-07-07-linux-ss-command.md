# SS Command



常见命令

```sh
ss -tnlp
```





```sh
# tcp
-t
# udp
-u
# numberic: don't try to resolve the service names
-n
# processes: show process using socket
-p
# Display only listening sockets
-l
# extended: show detailed socket information
-e
# options: show timer information
-o
# ipv4
-4
# ipv6
-6
```
