# calculator memory operation

mc m+ m- mr

## operators

- mc: memory clear 
- mr: memory recall
- ms: memory store
- m+: memory add
- m-: memory minus

**Descriptions**

默认M（内存）为0。

M+：功能是从计算器的记忆区中加上当前显示的数值，并将结果存回记忆区。

M-：功能是从计算器的记忆区中减去当前显示的数值，并将结果存回记忆区。

MR：随时点开查看内存

MS：存储当先显示的数值（覆盖之前）

MC：清空内存数值（置0）

**For example**

新打开的计算器，依次对对`11`使用`m+`，对`3`使用`m+`，对`13`使用`m-`

结果为`0+11+3+(-13)=1`，点击`mr`可以验证结果。
