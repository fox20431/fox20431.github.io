# 换行符

## Type

### CR(Carriage Return)
翻译：回车

### LF(Line Feed)
翻译：换行

### CRLF(Carriage Return Line Feed)
翻译：回车换行
| 缩写 | ASCII转义 | 系统                | ASCII值 |
| -    | -         | -                   | -       |
| CR   | \r        | MacIntosh(早期mac)  | 13      |
| LF   | \n        | Unix/Linux/Mac OS X | 10      |
| CRLF | \r\n      | Windows             |         |

## 野史

老式机械打印机
LF会将纸张上移一行，但水平位置没有改变  
CR则会使打字机托架滚回到纸张最左侧
