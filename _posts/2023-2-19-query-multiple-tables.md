# 多表查询

```sql
SELECT * FROM <表1> <表2>
```

上述结果为表1与表2的笛卡尔积。

举例：

```sql
SELECT * FROM students
```

```sql
from student A,student B,student C
where A. 姓名 = B. 姓名 and B. 姓名 = C. 姓名
```
